# Algoritmo CYK

Para executar o nosso programa, é necessário que ele receba como entrada um arquivo texto que contenha as produções da GLC G na FNC, também receberá uma palavra p que será derivada a partir da variável S. Terá como saída um arquivo texto que terá a matriz triangular produzida pelo algoritmo de CYK.

• A chamada deverá ser feita da seguinte forma:

	> ./Main glc.txt aaabbb saida.txt
    
Onde glc.txt é o arquivo de entrada com as produções de GLC na FNC.

aaabbb é a palavra p a ser derivada a partir de S.

saida.txt é o arquivo de saída que terá a matriz triangular produzida pelo algoritmo de CYK
